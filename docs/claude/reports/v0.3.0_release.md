## Challenges & Solutions

### Challenge 1: Responsive Grid Width
**Problem:** react-grid-layout requires fixed width, but we need responsive behavior

**Solution:**
- Container ref with `useEffect` to track width
- Debounced resize listener for performance
- Dynamic width recalculation on window resize

```typescript
const [containerWidth, setContainerWidth] = useState(1200);
useEffect(() => {
  const handleResize = debounce(() => {
    setContainerWidth(containerRef.current.offsetWidth);
  }, 100);
  window.addEventListener('resize', handleResize);
  return () => window.removeEventListener('resize', handleResize);
}, []);
```

### Challenge 2: Window State Persistence
**Problem:** Need to save/restore complex window layouts

**Solution:**
- Serialize WindowInstance[] to localStorage
- JSON-compatible config objects
- Workspace metadata (id, name, timestamps)
- Load on demand from menu

### Challenge 3: Type Safety for Dynamic Windows
**Problem:** Different window types need different configs

**Solution:**
- Discriminated union types for WindowConfig
- Type guards in window rendering
- Comprehensive TypeScript interfaces

```typescript
type WindowConfig =
  | { type: 'scanner'; config: ScannerConfig }
  | { type: 'news'; config: NewsConfig }
  | { type: 'alerts'; config: {} }
```

### Challenge 4: Playwright MCP Browser Version Mismatch
**Problem:** MCP server expected older Chromium version (1179) vs installed (1194)

**Solution:**
- Install Playwright locally in project
- Update `.mcp.json` to use local installation
- Configure with `node_modules` path
- Browsers now match MCP server version
